Geoffrey

A private household SMS assistant for grocery list management and everyday tasks.

What is Geoffrey?

Geoffrey is a private household SMS assistant operated by Andrew Dennis. The service allows pre-authorized household members to manage a shared grocery list and get assistance with everyday household tasks via text message.

How It Works

Authorized household members text the Geoffrey number with requests such as adding or removing grocery items, checking the current list, or asking general household questions. Geoffrey responds via SMS.

  • Add or remove items from the shared grocery list
  • View the current list at any time
  • Ask general household questions

Opt-In & Authorization

SMS Consent

This service is strictly limited to household members personally authorized by Andrew Dennis. There is no public sign-up. Authorization is granted directly and individually by the operator.

Each authorized user must provide explicit consent before receiving SMS text messages from Geoffrey. Consent is obtained by the operator sending a clear opt-in message that identifies the service, describes the SMS text messages to be received, and includes opt-out instructions. The user must reply YES to confirm consent before any messages are sent.

To opt out at any time, reply STOP to any message. You will receive no further SMS text messages after opting out. Reply HELP for assistance.

Message Frequency

Message frequency varies based on household activity. Messages are sent only in response to user-initiated requests or household events. Standard message and data rates may apply.
